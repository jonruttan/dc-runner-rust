# Library Release Policy

`data-contracts-library` publishes canonical shared bundle content through `data-contracts-bundles`.

Current release pins:

- `data-contracts-library-core` version `1.0.0`
- `data-contracts-library-runner-behavior` version `1.0.0`
- `data-contracts-library-bundle-tooling` version `1.0.0`
- `data-contracts-library-overlays` version `1.0.0`

Consumer repositories must pin exact bundle versions in lock/manifests.
